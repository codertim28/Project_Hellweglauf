1 Wettkampf

Die Wettkampffunktionalität ist die Kernfunktionalität dieser Software. Es gibt die Möglichkeit, Wettkämpfe auf Zeit oder Distanz durchzuführen.

1.1 Wettampf vorbereiten

Ist die "Wettkampfansicht" geöffnet, so kann der Wettkampf direkt gestartet werden. Allerdings können über den "Vorbereiten"-Knopf (oben rechts, Checklistensymbol) Änderungen an dem geöffneten Wettkampf vorgenommen werden.
Wird dieser Button geklickt, öffnet sich ein Modalfenster. Dieses beitet zum einen die Möglichkeit, Einstellungen wie Rundenlänge, Wettkampfdauer usw. einzustellen, zum anderen kann hier kontrolliert werden, welcher Schüler an dem Wettkampf teilnimmt und welcher nicht (Punkt 2 Schüler aufrufen). Dazu muss lediglich in das Textfeld geklickt und der Chips eingescannt werden. Unter dem Textfeld erscheint der Name des Schülers und es kann auf Ja (Schüler nimmt teil) oder Nein (Schüler nimmt nicht teil) geklickt werden.
Nimmt ein Schüler nicht teil, so wird dieser auch nicht gestartet (bekommt nicht die Runde 0 gesetzt beim Start des Wettkampfes).

1.3 Speichern und laden

Ist ein Wettkampf geöffnet, so kann dieser über den Menüpunkt "Datei" gespeichert werden. Anmerkung: Ein Wettkampf besteht aus zwei Dateien: wettkampfname.xml und wettkampfname.chips.xml

Ein gespeicherter Wettkampf kann über den Menüpunkt "Datei" geöffnet bzw. geladen werden. Dazu muss lediglich die Datei wettkampfname.xml ausgewählt werden. Das Programm lädt die wettkampfname.chips.xml automatisch.
